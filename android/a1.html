Practical 1: Setup Android Studio

Question
1.To study Android architecture and set up the Android development environment using Android Studio.
Java / XML 

        
B) Setting Up Android Development Environment (Android Studio)

Steps to set up Android Studio:

1.Download and Install
Go to Android Studio official site
 and download the installer.
Run the installer and follow the on-screen instructions.
Select Standard installation for beginners.

2.Launch Android Studio
Open Android Studio after installation.
First-time setup: it will download SDK, tools, and emulator images.

3.Configure SDK
Go to File → Settings → Appearance & Behavior → System Settings → Android SDK
Install the latest SDK Platform and SDK Tools.

4.Set Up an Emulator
Go to Tools → AVD Manager → Create Virtual Device
Select a device (e.g., Pixel 5) and system image (e.g., Android 13).
Launch emulator to test apps.

5.Create First Android Project
Click New Project → Empty Activity → Next
Give project name, package name, and choose language (Java/Kotlin).
Click Finish to generate project structure.

6.Run the App
Connect a real Android device via USB (enable developer mode) or use the emulator.
Click Run → Run App (green play button). App will launch.
